## Convert-LNK2DEMXML
#### Description
Create VMware Dynamic Environment Manager (XML) configuration files based on existing shortcuts (LNK).

#### Getting started
Open PowerShell and run the following script.

```powershell
.\Convert-LNK2DEMXML.ps1
```
